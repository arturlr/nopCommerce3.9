# nopCommerce

nopCommerce is an open-source ASP.NET MVC e-commerce platform with MS SQL Server backend. It's a fully customizable shopping cart solution designed for merchants who need a flexible, enterprise-grade e-commerce system.

## Key Features
- Plugin-based architecture for extensibility
- Multi-store support
- SEO-optimized (search-friendly URLs, structured content)
- Product catalog management (physical and digital goods)
- Customer management and reviews
- Order processing and shipping integration
- Payment gateway integrations
- Discount and promotion system
- Multi-language and multi-currency support
- Vendor management
- Blog and news functionality
- Forum system

## Target Users
Merchants who have outgrown simpler systems and need a robust, scalable e-commerce solution with full source code access.
